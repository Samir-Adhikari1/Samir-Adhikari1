### Steps to Activate the Snake Graph

**1. Create the workflow file:**

In your profile repo (`Samir-Adhikari1/Samir-Adhikari1`), create:
`.github/workflows/snake.yml`

**2. Paste this workflow:**

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 0 * * *"   # Runs daily at midnight UTC
  workflow_dispatch:        # Allows manual trigger from Actions tab
  push:
    branches:
      - main

jobs:
  generate:
    permissions:
      contents: write
    runs-on: ubuntu-latest
    timeout-minutes: 10

    steps:
      - name: Generate github-contribution-grid-snake.svg
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - name: Push snake to output branch
        uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

**3. Run it:**
- Go to → **Actions** → **Generate Snake** → **Run workflow**
- After ~1 minute, the SVG appears in the `output` branch
- The snake will auto-update every night via the cron schedule

</details>
