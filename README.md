<summary><strong>Activation Guide</strong></summary>

<br/>

### Step 1 — Snake Animation

Create `.github/workflows/snake.yml` in this repository:

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:
  push:
    branches: [main]

jobs:
  generate:
    permissions:
      contents: write
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Go to **Actions → Generate Snake → Run workflow** to trigger it manually.

---

### Step 2 — WakaTime

1. Create an account at [wakatime.com](https://wakatime.com)
2. Install the [WakaTime VS Code Extension](https://marketplace.visualstudio.com/items?itemName=WakaTime.vscode-wakatime)
3. Enter your API key when prompted
4. Coding time begins tracking automatically

---

### Step 3 — Profile Summary Cards

No setup required — renders automatically from public GitHub data.

### Step 4 — Trophies

Render automatically. Higher-tier ranks unlock with increased contributions, stars, and followers.
